**Formula:** `templateNumber = (locationIndex % 7) + 1`

## üîç **CARA VERIFIKASI TEMPLATE**

Untuk melihat template apa yang digunakan suatu kota:

1. Buka file `cities.json`
2. Cari index kota (mulai dari 0)
3. Hitung: `(index % 7) + 1`
4. Hasil 1-7 adalah nomor template

**Contoh:**
- Arongan Lambalek (index 0): `(0 % 7) + 1 = 1` ‚Üí **income-focus**
- Bubon (index 1): `(1 % 7) + 1 = 2` ‚Üí **freedom-focus**
- Johan Pahlawan (index 2): `(2 % 7) + 1 = 3` ‚Üí **growth-focus**
- Kaway XVI (index 3): `(3 % 7) + 1 = 4` ‚Üí **multiple-income**
- Meureubo (index 4): `(4 % 7) + 1 = 5` ‚Üí **professional-focus**
- Pante Ceureumen (index 5): `(5 % 7) + 1 = 6` ‚Üí **community-focus**
- Panton Reu (index 6): `(6 % 7) + 1 = 7` ‚Üí **mission-focus**
- Samatiga (index 7): `(7 % 7) + 1 = 1` ‚Üí **income-focus** (kembali ke template 1)

## üí° **MANFAAT SISTEM TEMPLATE ROTATION**

1. **SEO-Friendly**: Menghindari duplicate content penalty dari Google
2. **Scalable**: Mudah tambah lokasi baru tanpa perlu tulis konten manual
3. **Diverse Messaging**: Setiap template menyasar angle persuasi berbeda
4. **Automated**: Template dipilih otomatis berdasarkan index
5. **Maintenance**: Update 1 template = update ~47 halaman sekaligus

---

## üìù **CATATAN PENTING**

- Urutan kota di `cities.json` menentukan template yang digunakan
- Jangan random shuffle urutan cities kecuali ingin ubah distribusi template
- Setiap template memiliki variasi hero title, subtitle, dan content di file `contentVariations.ts`
- Kedua halaman (Daftar & Pendaftaran) menggunakan sistem template yang sama
